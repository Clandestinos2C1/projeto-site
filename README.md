# 🎵 Tenho Mais Amigos Que Discos

## 📌 Sobre o Projeto

O projeto **Tenho Mais Amigos Que Discos** consiste no desenvolvimento de um site voltado ao universo musical, criado com o objetivo de oferecer uma experiência moderna, organizada e agradável para os usuários.

A proposta da plataforma é unir tecnologia e nostalgia, permitindo que os usuários explorem diferentes estilos musicais e valorizem a experiência dos discos físicos em um ambiente digital acessível e intuitivo.

O site foi desenvolvido pensando tanto na experiência visual quanto na usabilidade, buscando um design atrativo, navegação simples e boa organização dos conteúdos.

---

# 🎯 Objetivos do Projeto

O principal objetivo deste projeto é aplicar conhecimentos de desenvolvimento web na criação de uma plataforma funcional e visualmente agradável.

## Objetivos Específicos

- Desenvolver uma interface moderna e responsiva;
- Melhorar a experiência do usuário durante a navegação;
- Organizar conteúdos de forma clara e intuitiva;
- Aplicar conceitos de desenvolvimento front-end e back-end;
- Criar uma identidade visual relacionada ao tema musical;
- Exercitar práticas de estruturação e estilização de páginas web;
- Utilizar versionamento de código durante o desenvolvimento.

# 🎨 Interface e Design

A interface do projeto foi planejada para transmitir uma estética moderna e relacionada ao universo da música.

Foram utilizados elementos visuais, cores suaves e organização de layout inspirados na cultura vintage dos discos de vinil, buscando proporcionar uma navegação confortável e agradável.

Além disso, o projeto segue conceitos de **responsividade**, permitindo melhor adaptação em diferentes tamanhos de tela.

---

# 🎵 Proposta do Site

O site foi criado para proporcionar aos usuários uma experiência musical mais calma, nostálgica e imersiva, incentivando a valorização da música física através dos discos de vinil.

A proposta principal é mostrar que ouvir música pode ir além do consumo rápido das plataformas digitais, incentivando a apreciação de álbuns completos e a conexão cultural com a música.

---

# 👥 Público-Alvo

O projeto é direcionado para:

- 🎧 Apreciadores de música;
- 💿 Colecionadores de discos;
- 🕰️ Pessoas interessadas em cultura vintage;
- 👨‍🎓 Jovens e adultos que desejam conhecer a experiência dos discos físicos.

---

# 📱 Responsividade

O desenvolvimento do site foi realizado seguindo o conceito **Mobile First**, priorizando inicialmente dispositivos móveis e posteriormente adaptando a interface para telas maiores.
