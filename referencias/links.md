# 🔤 Google Fonts

O projeto utiliza fontes disponibilizadas pela plataforma Google Fonts para melhorar a identidade visual e a legibilidade do site.

## Fontes Utilizadas

### Poppins

Utilizada nos títulos.

🔗 Link:
https://fonts.google.com/specimen/Poppins

### Inter

Utilizada nos textos.

🔗 Link:
https://fonts.google.com/specimen/Inter

---

## 📥 Importação das Fontes

```html
  <link rel="preconnect" href="https://fonts.googleapis.com" />
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
  <link href="https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;600;700;800;900&family=Inter:wght@300;400;500&family=Playfair+Display:ital,wght@1,400;1,700&display=swap" rel="stylesheet" />
  <link rel="stylesheet" href="css/style.css" />
```
